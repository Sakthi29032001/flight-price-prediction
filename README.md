# Flight Price Prediction

A machine learning-based web application that predicts airline ticket fares using historical flight data. The system is deployed using Flask and provides real-time predictions through an interactive web interface.

---

## 📌 Project Overview

This project applies machine learning techniques to estimate flight fares using historical flight data. It includes data preprocessing, feature engineering, model training, and deployment using a Flask web application.

---

## 🚀 Features

- Flight fare prediction using Machine Learning  
- Interactive web interface built with Flask  
- Data preprocessing and feature engineering  
- Hyperparameter tuning using RandomizedSearchCV  
- Model evaluation using R² Score  

---

## 🛠️ Technologies Used

- Python  
- Flask  
- Scikit-learn  
- Pandas  
- NumPy  
- HTML  
- CSS  
- Bootstrap  

---

## 🧠 Machine Learning Model

- Random Forest Regressor  
- RandomizedSearchCV optimization  
- Feature importance analysis
